Metafore One — Discovery Engine

Setup & Usage Guide — No technical knowledge needed

What does this app do?

The Discovery Engine reads your business documents (Word files, PDFs, or text files) and automatically builds an interactive Knowledge Graph — a visual map showing all the key processes, roles, systems, and policies in the document, and how they connect to each other. You can click any bubble on the graph to read more about it.

1

Install Python (first time only)

The app needs Python installed on your computer. If you've done this before, skip to Step 2.

Download Python →

⚠️ During installation, make sure to tick the checkbox that says "Add Python to PATH" before clicking Install Now. If you miss this, the app won't start.
2

Open the project folder

You should have received a folder called Discovery engine. Open it so you can see the files inside. It will look something like this:

📁

Discovery engine /

START.bat  ·  STOP.bat  ·  HOW_TO_RUN.html  ·  discovery-engine/

3

Double-click START.bat

Find the file called START.bat and double-click it. A black window will appear — this is normal.

First run: it will download and install some packages automatically. This takes about 1–2 minutes. Do not close the window while it's working. You'll see it say "[OK]" at each step when it's ready.
Second run onwards: it starts in about 5 seconds.
If Windows asks "Do you want to allow this app to make changes?" — click Yes.
4

The app opens in your browser automatically

After a few seconds, your web browser will open with the app at http://localhost:8001. You're ready to go!

If the browser doesn't open by itself, open it manually and type http://localhost:8001 in the address bar.

How to use the app

📂

Upload a document

Drag and drop a Word (.docx), PDF, or text (.txt) file onto the upload area on the left side — or click browse to find it on your computer. You can upload more than one document at a time.

⚡

Extract the Knowledge Graph

Click the Extract Knowledge Graph button. The AI will read your document and build the graph. This takes 15–30 seconds — you'll see a spinner while it works.

🔵

Explore the graph

Each bubble (node) is a process, role, system, or policy from your document. Click any bubble to see its full description and the exact text from the document that it came from. Drag to move bubbles around. Scroll to zoom in and out.

🏗️

Generate Object Model

Click the Generate Object Model button in the top-right to get a suggested data model for the entities in your document. Useful for developers who want to build a database from the graph. Use the Copy button to copy the code.

What do the colours mean?

Process (Teal)

Business processes, workflows, and procedures — things that happen step by step.

Role (Orange)

People, teams, departments, and job titles — who does what.

System (Blue)

Software, platforms, tools, and databases — the technology involved.

Policy (Red)

Rules, regulations, compliance requirements, and standards.

Data Entity (Purple)

Documents, records, forms, and data types — information that flows around.

Event (Amber)

Triggers, milestones, notifications, and deadlines — things that happen at a point in time.

Something not working?

❌ "Python is not installed" message

Go to https://www.python.org/downloads/, download the latest version, and install it. During installation, make sure to tick "Add Python to PATH". Then double-click START.bat again.

❌ Browser opens but shows an error page

The server is still starting up. Wait 10 seconds and refresh the browser page (press F5 or Ctrl+R).

❌ "Extraction failed" message when uploading a document

This usually means the app couldn't reach the AI service. Check your internet connection and try again. If it keeps failing, contact the person who sent you the project.

❌ Package installation failed

Try right-clicking START.bat and choosing "Run as administrator", then try again.

🛑 How do I stop the app?

Double-click STOP.bat to shut the server down cleanly. Or just close the small black window titled "MetaforeOne Discovery — Server".
